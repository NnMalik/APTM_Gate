Place offline prerequisites here for NUC deployment without internet.

1) .NET 10 ASP.NET Core runtime (required)
   - aspnetcore-runtime-10.0.x-linux-x64.tar.gz
     Download from: https://dotnet.microsoft.com/download/dotnet/10.0
   - Place the .tar.gz directly in this prerequisites/ folder.

2) PostgreSQL 16 (required for offline DB install)
   - Put the COMPLETE set of .deb files in:  prerequisites/postgresql/
   - "apt-get download postgresql ..." is NOT enough -- it skips dependencies.
     Use the Docker one-liner in prerequisites/postgresql/README.txt to capture the
     full dependency closure (postgresql-16, postgresql-common, libpq5, ssl-cert, ...).
   - setup.sh installs every .deb in prerequisites/postgresql/ together.

3) Browser for Start/Finish kiosk (optional)
   - chromium*.deb files in this prerequisites/ folder (else snap/apt is tried).

If the NUC has internet during install, the script falls back to online install for
any of the above, so these files are only needed for fully-offline deployment.
